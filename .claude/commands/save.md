# 작업 저장

오늘 작업 내용을 저장하고 GitHub에 동기화합니다 (배포는 하지 않음).

## 수행할 작업

1. **현재 브랜치 확인**
   - `dev` 브랜치인지 확인
   - 아니면 `git checkout dev`로 전환

2. **PROJECT_CONTEXT.md 업데이트**
   - 오늘 변경한 내용을 "최근 변경 이력" 섹션에 추가
   - 날짜: 오늘 날짜
   - 변경 사항 요약

3. **Git 커밋 & 푸시 (dev 브랜치)**
   - `git add .`
   - `git commit -m "작업 저장: [변경 내용 요약]"`
   - `git push` (dev 브랜치로 푸시 → 배포 안됨, 동기화만)

## 참고

- dev 브랜치는 동기화용이라 배포되지 않습니다
- 배포가 필요하면 `/deploy` 명령어를 사용하세요
